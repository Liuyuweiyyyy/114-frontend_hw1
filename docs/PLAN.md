# GitHub 偵察機 - 製作計畫

大家好！今天我們要一起做一個很有趣的網頁，叫做「GitHub 偵察機」！

這個網頁可以做什麼呢？
就是當我們輸入一個 GitHub 叔叔或阿姨的名字，它可以幫我們找到那個人的照片和資料，就像偵探一樣厲害！

## 第一步：做搜尋的框框

要先做一個可以打字的地方，就像這樣：
[           輸入 GitHub 名字           ] [ 按下搜尋 ]

這樣我們才能輸入要尋找的人的名字。

## 第二步：連線去 GitHub 抓資料

就像打電話一樣，我們要打去 GitHub 的圖書館（API）
問他們：「可以把這個叔叔的資料給我嗎？」

GitHub 會告訴我們：
- 叔叔的長相（頭像）
- 叔叔叫什麼名字
- 叔叔住在哪裡
- 有多少人追蹤叔叔

## 第三步：把資料變成卡片

把拿到的資料變成一張美美的卡片，上面有：
- 頭像（照片）
- 名字
- 簡介
- 追蹤者數字

## 第四步：處理錯誤

如果找不到這個人怎麼辦？
我們要顯示一個提示，告訴大家「找不到這個小朋友」！
